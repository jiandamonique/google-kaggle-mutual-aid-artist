Built from lived experience
and open-source care

The Artist Mutual Aid Directory was born during the 2021 Kaggle/Google BIPOC Data Fellowship. The idea was simple: use data science to make it easier for BIPOC and LGBTQ+ artists to find the help that already exists.

The fellowship ended. The need didn't. Five years later, the project is being revived — same architecture, deeper roots.

"Each one, reach one. Each one, teach one."

🌱 The Vision

A zero-dependency, open-source resource directory that any artist can use, any community can fork, and any mutual aid network can contribute to.

🔬 The Method

Python · BeautifulSoup · Pandas → structured data → vanilla HTML/CSS/JS. No framework. No server. Zippable, forkable, lightweight by design.

👤 Who It's For

BIPOC artists. LGBTQ+ creatives. Anyone in the arts who needs housing, healthcare, legal help, or emergency funds — and doesn't know where to start.

🤝 Built By

Jianda Monique — Jr. Data Scientist, author, musician, and multidisciplinary artist. 2021 Kaggle/Google BIPOC Data Fellow. Mentored by Prasun Kumar Mishra and Julia Elliott.
